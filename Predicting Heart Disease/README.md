# Создадим модель многоклассовой классификации для предсказания признаков болезни сердца у пациента, в зависимости от его физических характеристик (возраст, пол, холостерол и пр.).

В базе данных содержиться 14 атрибутов о физическом тестировании пациентов:

-age - возраст;
-sex - пол;
-cp - chest pain - тип боли в груди;
-trestbps - resting blood pressure - давление в состоянии покоя;
-chol - уровень холестирирна в крови мг/дл;
-fbs - fasting blood sugar -уровень сахара в крови > 120 мг/дл;
-restecg - resting electrocardiographic result - результаты электрокардиограммы (значения 0, 1, 2);
-thalach - максимальный пульс;
-exang - exercise induced angina - возникновение ангины в результате упражнений;
-oldpeak - пиковые значения электрокардиограммы;
-slope - наклон пикового значения в электрокардиограмме;
-ca - количество крупных сосудов (0, 1, 2, 3), окрашенных флурозопией;
-thal - 3 = нормально, 6 = фиксированный дефект, 7 = обратимый дефект;
-target - указывает на наличие заболеваний сердца: 0 - нет заболевания и 1 - есть заболевание;

# Результат
f1-score 0.84. Для класса 0 вероятность 99%, а для класса 1 веротность 0,0000001%
